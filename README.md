# Automate-Python
Projet Python d’implémentation d’automates finis:déterminisation, complétion, complémentaire, intersection, union, concaténation et étoile de Kleene. Le projet inclut également une application permettant de parcourir un fichier texte et d’en extraire tous les nombres décimaux grâce à un automate. Développé dans le cadre du UE Mathématique discrète.
